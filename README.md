## Texter

This is a code editor built with Kotlin, using the Swing and AWT libraries.

#### Features:
- Syntax highlighting for several languages
- Multiple files can be opened in separate tabs
- Multiple themes

##### Syntax highlighting is available for:
 - C
 - Css
 - C#
 - Rust
 - GDScript
 - Html
 - Java
 - Kotlin
 - Makefile
 - Markdown
 - Python
 - Shellscript
 - Xml
